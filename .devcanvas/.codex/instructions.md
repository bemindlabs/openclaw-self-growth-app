# Codex Instructions

## Project Context

This workspace is managed by AI-DLC (AI-Driven Development Life Cycle).

## Working with This Project

- Check `.scrum/project.md` for project metadata
- Check `.scrum/stories/` for current work items
- Check `.scrum/dlc/state.md` for current development phase
- Check `agents/` for available agent profiles and their specializations

## Development Workflow

1. Check current DLC phase in `.scrum/dlc/state.md`
2. Pick a story from `.scrum/stories/` matching the current phase
3. Implement changes following the project conventions
4. Update story status when complete
5. Commit with conventional commit message

## Conventions

- Follow the existing code style
- Write meaningful commit messages
- Keep changes focused and reviewable
